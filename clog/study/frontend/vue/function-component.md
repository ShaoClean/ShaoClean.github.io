# Vue3函数式组件（不使用模板）

## 1.核心思路

使用纯js生成组件核心思路其实就只有两步：

- 1.创建虚拟dom（vnode）
- 2.渲染成真实节点（render）

实现这这两步需要用到两个核心函数:
- `h()函数`：

    `h()`函数用于生成一个虚拟dom（vnode），详情见[h()函数](./h.md)

- `render()函数`：

    `render()`函数用于渲染虚拟dom，将虚拟dom变成真实dom放置到页面指定节点。它接收两个参数：

    - 第一个参数：vnode ｜ component，虚拟节点或`.vue`文件
    - 第二个参数：需要挂载的节点
