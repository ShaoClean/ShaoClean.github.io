删除数据库中n行到m行到记录：

下面是删除数据库中30-100行记录的sql语句

```sql
delete from image_info where id in (select id from (SELECT * FROM image_info limit 30,100) as t);
```

本来以为是非常简单的事情，结果话费了我半个多小时
